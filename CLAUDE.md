# Factor-Craft 世界 — 开发指南

本文档为 Factor-Craft 世界观仓库的 AI 协作指南。

## 仓库定位

**这是纯世界观/文学创作仓库**，不涉及任何 mod 代码实现。所有内容服务于：
- 世界观设定完善
- 历史细节填充
- 碎片化叙事创作
- 文学/故事创作

## 协作原则

### 叙事真实性分级

文档中的事件、人物引用采用可靠性分级标记：

| 标记 | 含义 |
|------|------|
| **高** | 多方化石/文献交叉验证，几乎确定 |
| **中** | 单源记录或部分推断 |
| **低** | 传说色彩浓厚，可能存在虚构 |
| **※** | 学界争议焦点 |
| **△** | 传说人物，可能不存在 |

### 碎片化叙事原则

- 所有"历史文献"必须标注来源层级（化石层/口述层/传说层/考古层/当代层）
- 不同时代的叙事者有不同的认知局限，不能以"全知视角"叙述
- 真相被层层包裹，最终揭示需要铺垫

### 纪年法

| 纪元 | 纪年法 | 示例 |
|------|--------|------|
| 第一纪元 | FC（Factor Calendar） | FC 4500 |
| 第二纪元 | 2E | 2E 2500 |
| 第三纪元 | 3E | 3E 2200 |
| 第四纪元 | 4E | 4E 2480 |

### 禁止事项

- ❌ 不要在世界观文档中引入 mod 实现细节（如物品ID、方块名称）
- ❌ 不要在文学创作中泄露"全知视角"的真相，除非符合叙事框架
- ❌ 不要在未确认设定前强行推进剧情线

## 文件命名

- 所有文件使用中文命名
- 文档使用 `.md` 格式
- 碎片文献放在 `碎片文献/` 目录下，按层级分类

## 创作状态标记

在文档开头使用状态标记：

```
## 状态

- [x] 已完成
- [ ] 进行中
- [ ] 待填充
- [ ] 框架完成
```
