# 因子与世界本质

> **版本**：v1.0  
> **说明**：Factor-Craft 世界观核心设定文档

---

## 一、核心理念：因子单一概念

**因子（Factor）** 是宇宙暗能量的可见形态，是所有物质、能量、生命的基础构成单位。

```
虚空（不可见、弥漫所有维度）
     ↓ 维度裂隙装置抽取
因子（Factor，可操控的能量粒子）
     ↓ 按浓缩程度分级
各种应用：能源、医疗、建筑、维度操控
```

**关键点**：
- 因子是**单一能量形式**，浓缩程度决定特性和用途
- 原初文明掌握了将虚空能量凝结为因子的技术
- 因子的特性取决于**浓缩程度和编织方式**，而非"类型"

---

## 二、虚空与因子

### 2.1 虚空（Void / 原源质海）

虚空是弥漫所有维度的暗能量场，无法被直接利用。原初学者发现可以通过**维度裂隙装置**从虚空中抽取能量并凝结为因子。

| 属性 | 说明 |
|------|------|
| 分布 | 渗透所有维度 |
| 可见性 | 不可见 |
| 可利用性 | 需要裂隙装置转化 |
| 总量 | 近乎无限 |
| 原名 | "原源质海" |

### 2.2 因子（Factor）

因子是虚空能量的可操控形态。

| 属性 | 说明 |
|------|------|
| 来源 | 从虚空中抽取并凝结 |
| 存在形态 | 可浓缩、可编织、可转化 |
| 宇宙原型 | 宇宙暗能量 |
| 原初名称 | "因子本相" |

## 三、因子等级体系

原初文明建立了因子等级标准（T1-T5 + TΩ）：

| 等级 | 名称 | 说明 | 典型用途 |
|------|------|------|----------|
| T1 | 基础级 | 直接从环境提取 | 日常能源 |
| T2 | 精炼级 | 第一次提纯 | 机器运转 |
| T3 | 浓缩级 | 高纯度浓缩 | 高级合成 |
| T4 | 纯净级 | 几乎无杂质 | 精密设备 |
| T5 | 本源级 | 接近属性本源 | 维度技术 |
| TΩ | 创世级 | 理论最高级 | 仅存于理论 |

### 等级体系的原初视角

```
T1-T4: 工业级应用（广泛民用）
T5: 军用级/特殊用途
TΩ: 仅限议会核心实验室
```

## 四、因子应用

### 4.1 自然分布

因子天然存在于世界中，浓度因区域而异：

| 维度 | 因子浓度 | 原因 |
|------|----------|------|
| 主世界 | 低-中 | 裂隙装置残骸泄漏 |
| 下界 | 高 | 第一次灾变时倒灌最严重 |
| 末地 | 极高 | 裂隙源头所在地 |

### 4.2 因子潮汐

因子浓度随时间周期性波动，称为**潮汐**。

- **技术解释**：损坏的裂隙装置残余运作
- **当前文明解释**：世界的呼吸，完全自然

### 4.3 因子畸变

高浓度因子接触普通物质会导致**因子畸变**——生物变异、矿物暴走。这是第一次灾变的产物。

## 五、维度裂隙装置

### 5.1 装置原理

维度裂隙装置是原初文明的核心技术，用于从虚空中抽取能量并凝结为因子。

```
虚空能量 → 裂隙装置 → 因子凝结 → 各种应用
```

### 5.2 五大裂隙装置（归本计划核心）

原初文明共建造了五座核心裂隙装置，位于不同地点：

| 装置 | 位置 | 状态 | 结局 |
|------|------|------|------|
| 第一座 | 主世界东部 | 4E 2468 关闭 | 成功关闭 |
| 第二座 | 主世界北部 | 4E 2478 爆炸 | 爆炸损毁 |
| 第三座 | 圣城地下 | 4E 2483 关闭 | 冲突中关闭 |
| 第四座 | 裂隙城废墟 | 4E 2484 关闭 | 成功关闭 |
| 第五座 | 中央穹顶深层 | 4E 2498 逆向转化 | 源初以身殉道 |

### 5.3 归本计划的真相

**归本计划**是第一纪元因子议会推动的终极项目——将因子逆向转化回虚空。议会认为这将为文明打开"通向无限能源的大门"。

实际上，这是导致第一次灾变的直接原因。

## 六、机器设备的原型

### 6.1 模组机器 → 原型对照

| 模组机器 | 原初原型 | 功能 |
|----------|----------|------|
| 因子提取器 | 维度虹吸器 | 从裂隙中提取因子 |
| 因子合成器 | 因子重组台 | 按需合成物质 |
| 因子储罐 | 维度稳定容器 | 存储因子 |
| 因子监控器 | 维度监控终端 | 查看浓度 |

### 6.2 技术差距

现代机器是原初遗迹的仿制品：
- 效率约为原型的 30%
- 但更稳定（原初机器在灾变中已损坏）

## 七、关键文献：化石

原初文明的完整科技记载称为**《因子化石》**，共 1,000 片。

当前状态：
- 约 150 片存在于各学者团体
- 50 片下落不明
- 研究系统（Research）实际上是破译这些碎片的过程

## 八、世界背景

### 远古时代

原初文明掌握了因子技术，建立了辉煌的科技文明。但因议会推动"归本计划"——将因子逆向转化回虚空——导致维度边界崩塌，第一次灾变爆发，文明毁灭。

### 现代世界

玩家作为探索者，在这个世界中重新发现因子的使用方法。你可以从自然中提取因子，可以合成更高级的因子，可以使用因子创造物品、建造机器、释放能力。
