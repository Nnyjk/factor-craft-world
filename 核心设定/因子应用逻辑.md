# 因子应用逻辑

> **版本**：v1.0  
> **说明**：基于"因子守恒"与"潮汐循环"的底层应用逻辑推演

---

## 底层原则

### 因子守恒

因子在使用中**总量不变**，仅发生形态转化：

```
虚空（不可穷尽）
    ↑ 抽取           ↓ 逆向转化（终局能力）
因子（总量守恒）
    ↑ 萃取           ↓ 归本
材料 ←→ 物品 ←→ 能量
```

**正常操作中不存在"消耗"**——因子在使用后回归环境，只是形态/位置改变。

### 潮汐循环

因子浓度随时间周期性波动，称为**潮汐**：

| 阶段 | 浓度 | 表面解释 |
|------|------|----------|
| 枯竭 | 0-20% | 世界沉睡 |
| 低能 | 20-40% | 蓄力期 |
| 稳定 | 40-60% | 平衡期 |
| 高能 | 60-80% | 世界觉醒 |
| 过载 | 80-100% | 危险临界 |

**潮汐真相**：裂隙装置的残余运作导致因子周期性释放。因子的总量不变，但在位置和浓度上循环波动。

## 四大职业的应用闭环

### 萃取师（Extraction）

**职责**：从环境提取因子，使其从游离态凝结为可用态

**核心操作**：

| 操作 | 说明 |
|------|------|
| **抽取** | 从高浓度区域/裂隙提取游离因子 |
| **浓缩** | 提高因子的密度/纯度 |
| **附着** | 将因子固定在材料上（浸入深度 = T级） |

**T级的物理意义**：仅指因子在材料中的**附着密度/浸入深度**，不指"类型"。

| 等级 | 浸入程度 | 说明 |
|------|----------|------|
| T1 | 浅层附着 | 普通材料，表面浸入 |
| T2 | 中层附着 | 金属，因子渗透至结构内部 |
| T3 | 深层附着 | 稀有矿物，因子与晶体结构结合 |
| T4 | 近乎饱和 | 特殊合金，因子充满微观间隙 |
| T5 | 完全浸入 | 维度材料，因子与材料融为一体 |

---

### 编织师（Weaving）

**职责**：改变因子的**组织形态**，不改变总量

**编织的本质**：对已凝结的因子进行"再组织"，只改变因子的存在方式，不消耗因子本身。

**核心操作**：

| 操作 | 说明 | 示例 |
|------|------|------|
| **重组** | 改变因子的编织结构 | 松散附着 → 紧密编织 |
| **塑形** | 改变存在形态 | 游离态 ↔ 结晶态 ↔ 附着态 |
| **导向** | 改变因子的流动方向 | 聚集/扩散/定向传输 |
| **共振** | 使多个因子产生协同效应 | 放大/抑制/同步 |

**关键约束**：
- 编织只能操作"已被萃取凝结"的因子
- 编织改变的是因子的**组织方式**，不是因子本身
- 编织不改变因子总量

**不存在元素属性**：编织产生的效果差异来自于"编织密度/结构"不同，而非因子有不同的"属性"。

---

### 守望者（Vigil）

**职责**：感知、勘探、记录——不改变因子，只观察

**当前时代（第四纪元）的实际职能**：

| 职能 | 说明 | 表面理由 |
|------|------|----------|
| **勘探** | 寻找高浓度因子区域、矿脉 | "寻找先祖遗留的能量点" |
| **感知** | 感知因子流动方向、浓度变化 | "感应世界的呼吸" |
| **监测** | 记录潮汐规律、异常波动 | "记录自然的节律" |
| **预警** | 预判因子聚集/风暴 | "预测危险的来临" |

**守望者的局限**：他们不知道自己监测的其实是裂隙装置的运作状态。他们以为自己在研究"自然现象"。

---

### 裂隙师（Rift Worker）

**职责**：直接操作裂隙装置——最高风险、最高回报

**核心操作**：

| 操作 | 说明 |
|------|------|
| **抽取控制** | 调节从裂隙抽取因子的速率 |
| **释放调节** | 控制因子向环境的释放节奏 |
| **泄漏封堵** | 修补裂隙的异常泄漏 |

**职业闭环**：

```
萃取师 ← 原料供给 ← 裂隙师
    ↓
编织师 ← 形态加工 ← 萃取师
    ↓
守望者 ← 监测预警 ← 编织师（使用后因子回归环境）
    ↑
裂隙师 ← 补充原料 ← 萃取师
```

因子在这个闭环中**总量守恒**，在不同职业间流转转化。

## 消耗：认知之外的终局能力

### 认知层级

| 层级 | 对"消耗"的理解 |
|------|----------------|
| **普通人** | 不知道因子会消失，认为"用了就用了"，因子回归环境 |
| **学者** | 知道因子守恒，能解释"用了之后因子去哪了"——回归环境/材料 |
| **深层学者** | 知道"归本计划"会导致因子消失，但不清楚具体机制 |
| **源初** | 唯一知道如何触发"不可逆消耗"的人 |

### 为什么没有人知道"消耗"？

在正常操作中，**因子根本不会消失**：

- 用因子点灯 → 因子转化为光能 → 仍然存在于环境中
- 用因子建造 → 因子附着在材料上 → 仍然在
- 用因子战斗 → 因子释放能量 → 仍然弥散在空气中

**因子的"消失"只存在于理论中**——TΩ级操作、归本计划、源初的牺牲。普通人连想象"因子可以消失"的基础都没有。

### 源初的特殊性

源初能"听到因子的声音"，这意味着他能感知到因子在**正常循环之外**的路径——即逆向转化回虚空的可能性。

这不是"更强的守望者"，而是一种**完全不同的感知维度**：
- 守望者感知：因子在哪里、流向哪里、浓度多少
- 源初感知：因子可以**去哪里**（包括不应该去的虚空）
