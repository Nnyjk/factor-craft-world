## 核心设定

### 因子（Factor）

**因子**是宇宙暗能量的可见形态，是所有物质、能量、生命的基础构成单位。

- **虚空**：弥漫所有维度的暗能量场，无法直接利用
- **因子**：从虚空中抽取凝结的可操控能量粒子
- **等级体系**：T1-T5 + TΩ（创世级）

### 三纪元框架

| 纪元 | 名称 | 时间 | 文明状态 |
|------|------|------|----------|
| 第一纪元 | 创世者纪元 | FC 1 - FC 4500 | 原初文明，因子技术巅峰，因第一次灾变毁灭 |
| 第二纪元 | 灰烬纪元 | 2E 1 - 2E 4500 | 继承者文明，灾变后重建，因第二次灾变重创 |
| 第三纪元 | 新生纪元 | 3E 1 - 3E 2500+ | 当前文明，自称"第一代"，不知前两次灾变 |
| **第四纪元** | **终局纪元** | **4E 1 - 4E 2520** | **因子消失，循环终结，完整闭环** |

### 循环的终结（第四纪元终局）

第四纪元终局于 **4E 2520**，核心事件：
- 源初以自身为媒介，将因子彻底逆向转化回虚空
- 所有裂隙装置关闭，维度边界闭合
- 因子从世界上永远消失
- 旧世界秩序崩溃，新文明在废墟上重建

> *"不要再建造了。去生活。"* —— 源初

## 叙事结构：碎片化叙事

世界历史因灾变而严重损毁，叙事采用五层碎片结构：

1. **化石层**（第一纪元科学记载）
2. **口述层**（避难所幸存者的恐惧记忆）
3. **传说层**（第二纪元的神话化叙事）
4. **考古层**（探险家的困惑笔记）
5. **当代层**（第三纪元的误解与遗忘）
