# Factor Craft 派系志

> **说明**：以下派系基于历史人物和事件派生，按活跃时期排列。带"※"标记的为隐秘组织，带"†"标记的为已消亡派系。

---

## 第一纪元派系

### 因子议会（Council of Origin）

| 项目 | 内容 |
|------|------|
| **存续时间** | FC 50 至 FC 4500 |
| **总部** | 中央穹顶（已毁） |
| **意识形态** | 知识进步、技术万能、议会民主 |
| **政治立场** | 中央集权、技术精英统治 |
| **结局** | 第一次灾变中彻底覆灭 |

**派系概述**：

因子议会是第一纪元的最高权力机构，由各大研究部门的代表组成。它是因子技术发展的产物，也是因子技术滥用的推动者。

议会内部逐渐形成两个派系：

| 派系 | 领袖 | 立场 |
|------|------|------|
| **发展派** | 卡尔丁·裂隙 | 主张扩大因子应用，推进归本计划 |
| **保守派** | 奥伯恩·反向（精神领袖） | 主张限制因子扩张，关注安全风险 |

**核心矛盾**：

议会相信"知识没有尽头"，却忽视了"知识的力量必须有其边界"。这种集体傲慢最终导致了议会对奥伯恩警告的系统性忽视——不是因为警告没有道理，而是因为"进步的声音压过了谨慎的声音"。

**遗产**：

- 议会的大量技术档案成为第二、三纪元"遗迹"的基础
- 议会的失败为后世提供了深刻的反面教材（但是否被吸取？）
- 议会的一些成员（如守望部门的无名创始人）试图留下警告，但未能改变结局

### 守望部门（Vigil Department）

| 项目 | 内容 |
|------|------|
| **存续时间** | FC 50 至 FC 4500 |
| **总部** | 维度监控中心（已毁） |
| **意识形态** | 维护维度稳定、监控系统安全 |
| **政治立场** | 技术中立、远离政治 |
| **结局** | 第一次灾变中总部被毁，部分成员进入避难所 |

**派系概述**：

守望部门是四大部门中最"低调"的一个。它的职责是监控维度边界的稳定性，维护裂隙装置的安全运行。

然而，守望部门逐渐发现自己的"监控职责"与议会主流的"发展至上"路线产生了冲突。赛琳娜·因子的事件是这种冲突的典型体现——守望部门的警告被议会轻描淡写地驳回。

**特殊遗产**：

守望部门是唯一一个在灾变后仍然保留了**系统化监控日志**的部门。这些日志后来成为第二纪元"深层学者"研究维度边界问题的基础。

## 第二纪元派系

### 拾荒者议会（Scavenger Council）

| 项目 | 内容 |
|------|------|
| **存续时间** | 2E 600 至 2E 4500 |
| **总部** | 裂隙城（已毁） |
| **意识形态** | 遗迹开发、技术实用主义、利益至上 |
| **政治立场** | 城邦联盟、商人寡头政治 |
| **结局** | 第二次灾变中大部分城邦被毁，议会瓦解 |

**派系概述**：

拾荒者议会是第二纪元的最高权力机构，由各大城邦的"拾荒者行会"代表组成。它与第一纪元的因子议会在名称上有传承关系，但性质完全不同——因子议会是**知识精英统治**，拾荒者议会是**利益精英统治**。

议会的核心信念是：遗迹是"先祖的遗产"，开发遗迹是"文明的义务"。在这种信念下，议会推动了大规模遗迹开发，甚至重启了一些原纪元的裂隙装置。

**※ 关键讽刺**：

拾荒者议会中没有人真正理解裂隙装置的原理。他们只知道"这玩意儿能产出因子"，却不知道"这玩意儿失控会毁灭世界"。

**与深层学者的冲突**：

当深层学者提出"关闭裂隙装置"的主张时，拾荒者议会的第一反应是："关闭了，我们靠什么发电？"——这种实用主义立场最终导致了学术管制法的颁布和深层学者的被迫害。

### 宗教正统（Religious Orthodoxy）

| 项目 | 内容 |
|------|------|
| **存续时间** | 2E 250 至 2E 4500（至今仍有影响） |
| **总部** | 第一圣殿（已毁，部分转移） |
| **意识形态** | 天罚叙事、神学解释、信仰维护 |
| **政治立场** | 宗教神权、反对学术质疑 |
| **结局** | 第二次灾变中总部被毁，但信仰体系延续至第三纪元 |

**派系概述**：

宗教正统是基于《第一圣典》建立的宗教体系。它将第一纪元的科学成就解读为"神迹"，将灾变解读为"天罚"，将遗迹解读为"神的遗产"。

这个体系的建立者焚典·学者可能是出于好意——他认为，在第二纪元的技术水平下，让普通人知道"灾变是技术失误"只会导致他们拒绝使用一切技术，从而文明倒退。

然而，这种"保护性无知"的政策最终压制了真正的学术探索，使得第二纪元重蹈了第一纪元的覆辙。

**核心教义**：

| 教义 | 宗教解释 | 技术真相 |
|------|----------|----------|
| 因子的起源 | 神创的能量 | 原初文明从虚空抽取 |
| 因子潮汐 | 神的呼吸 | 损坏的裂隙装置运作 |
| 灾变原因 | 天罚（先觉者傲慢） | 归本计划失控 |
| 遗迹的意义 | 神的遗产 | 第一/二纪元的废墟 |
| 正确态度 | 敬畏、使用、不追问 | 追问可能避免下一次灾变 |

**教宗·天罚的遗产**：

他的压制政策虽然在短期内维护了宗教权威，但也加剧了深层学者与宗教势力的对立。当灾变再次来临时，宗教正统的信誉扫地。

### 探源院（Institute of Origin）

| 项目 | 内容 |
|------|------|
| **存续时间** | 2E 2500 至 2E 4500 |
| **总部** | 探源院总部（已毁） |
| **意识形态** | 学术研究、真相探索、技术进步 |
| **政治立场** | 学术独立、与政治保持距离 |
| **结局** | 第二次灾变中被毁，但核心成员转入地下 |

**派系概述**：

探源院是第二纪元最重要的学术机构。它的成立标志着第二纪元开始从"纯粹使用遗迹"转向"理解遗迹原理"。

探源院的第一任院长是一个务实的技术官僚，他的目标是"让第二纪元不再依赖偶然的发现，而是系统性地发展技术"。然而，他的继任者们逐渐发现，理解遗迹原理不可避免地指向了"灾变的真相"。

**探源·院长的发现**：

当院长发现"灾变不是天罚，而是技术失控"时，他面临了一个选择：

- **选择一**：公开发现 → 可能导致技术拒绝狂潮，文明倒退
- **选择二**：隐瞒发现 → 文明可能在不知情的情况下重蹈覆辙
- **选择三**：秘密保存，等待时机 → 将选择权交给未来

他选择了第三个方案，并由此催生了"深层学者"组织。

### 深层学者（Deep Scholars） ※

| 项目 | 内容 |
|------|------|
| **存续时间** | 2E 3200 至今（第三纪元仍有后继者） |
| **总部** | 无固定总部（分散秘密据点） |
| **意识形态** | 真相保存、知识传承、预防灾变 |
| **政治立场** | 不参与政治、但必要时干预 |
| **结局** | 第二次灾变中核心成员大批死亡，后继者分散隐藏 |

**派系概述**：

深层学者是第二纪元最重要的隐秘组织。它由裂隙·守望创立，目标是"保存第一、二纪元的真相，寻找避免第三次灾变的方法"。

**组织结构**（第二纪元）：

```
深层学者
├── 核心圈（3-5人）
│   └── 掌握完整档案、知道所有秘密据点位置
├── 研究圈（20-30人）
│   └── 负责化石研究、维度监测、裂隙装置追踪
└── 外围圈（100+人）
    └── 散布在各城邦、担任各种职业、传递情报
```

**分裂危机**（第二纪元末期）：

在第二次灾变前，深层学者内部出现了严重的路线分歧：

| 派系 | 立场 | 主张 |
|------|------|------|
| **激进派** | 公开真相 | 不管后果，立即向公众公开灾变真相 |
| **稳健派** | 等待时机 | 保存实力，等待"能够承受真相的人"出现 |
| **行动派** | 技术干预 | 不等时机，现在就尝试关闭裂隙装置 |

学术管制法颁布后，激进派和行动派成员大批被清洗，稳健派勉强维持了组织的存续。

**※ 关键遗产**：

- 深层学者保存了第一、二纪元的核心文献
- 深层学者记录了仍在运作的裂隙装置的位置
- 深层学者留下了"第三次灾变前兆"的监测数据
- 深层学者的后继者在第三纪元仍在活动

## 第三纪元派系

### 探宝人行会（Treasure Hunters Guild）

| 项目 | 内容 |
|------|------|
| **存续时间** | 3E 800 至今 |
| **总部** | 各城邦分部，无中央总部 |
| **意识形态** | 遗迹开发、利益至上、职业自由 |
| **政治立场** | 商业中立、与政治保持距离 |
| **结局** | 仍在活跃 |

**派系概述**：

探宝人行会是第三纪元最大的遗迹开发组织。它的成员包括探险家、考古者、商人、雇佣兵等各种背景的人。

行会不关心遗迹的"来历"或"意义"，只关心"能卖多少钱"。这种纯粹的实用主义使他们成为遗迹开发的主力，但也使他们成为"真相探索"的最大阻碍——因为如果遗迹的来历被重新诠释，他们赖以生存的"遗迹=遗产"叙事就会崩溃。

**与深层学者的潜在冲突**：

如果探宝人发现了深层学者秘密保存的档案，可能引发两大阵营的正面冲突。

### 源质教会（Church of Origin）

| 项目 | 内容 |
|------|------|
| **存续时间** | 3E 300 至今 |
| **总部** | 无中央总部（各城邦教区自治） |
| **意识形态** | 先祖崇拜、天罚信仰、敬畏自然 |
| **政治立场** | 宗教权威、道德审判 |
| **结局** | 仍在活跃，但在动摇中 |

**派系概述**：

源质教会是第一纪元宗教正统在第三纪元的延续。它将"灾变是神的惩罚"作为核心教义，将"敬畏遗迹、使用遗迹、不追问遗迹"作为信仰实践。

然而，与第二纪元的宗教正统不同，第三纪元的源质教会面临着更严峻的挑战——因为第三纪元的遗迹开发规模远超第二纪元，而教会对此几乎毫无约束力。

**内部危机**：

随着越来越多的学者开始质疑"天罚叙事"，源质教会内部也开始出现分裂：

| 派系 | 立场 | 主张 |
|------|------|------|
| **正统派** | 坚守经典 | 《第一圣典》不容质疑，灾变就是天罚 |
| **改革派** | 重新诠释 | 天罚叙事有其道理，但需要与新的发现调和 |
| **怀疑派** | 动摇怀疑 | 《第一圣典》可能是错的，但我们该怎么办？ |

### 学者公会（Scholar Guild）

| 项目 | 内容 |
|------|------|
| **存续时间** | 3E 2050 至今 |
| **总部** | 各城邦学术中心 |
| **意识形态** | 知识探索、理性主义、技术进步 |
| **政治立场** | 学术中立、逐步参政 |
| **结局** | 仍在活跃，处于上升期 |

**派系概述**：

学者公会是第三纪元最重要的知识阶层组织。它与第一纪元的探源院有间接传承关系——一些学者的师承可以追溯到第二纪元的探源院学者。

学者公会的主要工作是"收集和解读化石碎片"。他们的解读方法比第二纪元的宗教正统更加科学化，但他们仍然受到"天罚叙事"的约束——因为完全否定《第一圣典》在政治上是不安全的。

**与深层学者的可能接触**：

学者公会中的一些成员可能已经意识到"有什么不对劲"，但缺乏关键信息。如果深层学者的后继者选择接触学者公会，可能触发真相探索的新浪潮。

### 深层学者后继者（Heirs of Deep Scholars） ※

| 项目 | 内容 |
|------|------|
| **存续时间** | 2E 4500 至今 |
| **总部** | 无固定总部（极其分散） |
| **意识形态** | 真相保存、预防灾变、谨慎行动 |
| **政治立场** | 不参与政治、绝对隐秘 |
| **结局** | 仍在活跃，但力量微弱 |

**派系概述**：

深层学者后继者是第一、二纪元深层学者组织的延续。他们散布在第三纪元的各个角落，各自掌握着不完整的档案碎片，通过隐秘的方式保持联系。

**※ 关键困境**：

与第二纪元的祖先不同，第三纪元的深层学者后继者面临一个前所未有的问题：**他们不知道"能够承受真相的人"是否真的存在。**

他们的前辈等待了两千多年。而他们现在观察到的情况是：因子浓度在持续上升，下界门越来越不稳定，一些"不应该启动"的原纪元设备开始自行启动……

**路线分歧**（第三纪元当前）：

| 派系 | 立场 | 主张 |
|------|------|------|
| **公开派** | 立即行动 | 时机已经成熟，必须立即公开真相并尝试关闭裂隙 |
| **等待派** | 继续等待 | 力量还不足以对抗主流，必须继续等待 |
| **培养派** | 培养觉醒者 | 找到那个"能够承受真相的人"，将他培养成改变世界的力量 |
| **放弃派** | 顺其自然 | 灾变可能无法阻止，应该专注于保存最后的文明火种 |

**※ 核心问题**：

- 他们掌握的档案碎片在哪里？
- 谁能证明自己是"真正的深层学者后继者"？
- 他们会如何对待"觉醒者"——是保护，还是控制？

## 派系核心矛盾

| 矛盾 | 双方 | 核心冲突 |
|------|------|----------|
| **真相 vs 稳定** | 深层学者 vs 宗教/政治势力 | 公开灾变真相可能引发混乱，但隐瞒真相可能导致重蹈覆辙 |
| **进步 vs 安全** | 发展派 vs 保守派 | 遗迹开发带来短期利益，但可能加速第三次灾变 |
| **知识 vs 信仰** | 学者公会 vs 源质教会 | 理性探索与宗教信条之间的根本张力 |
| **现在 vs 未来** | 行动派 vs 等待派 | 是立即行动阻止灾变，还是保存力量等待时机？ |
| **遗忘 vs 记忆** | 普通大众 vs 真相守护者 | 无知是幸福还是灾难的温床？ |

## 各派系对"因子潮汐"的理解

| 派系 | 理解 | 来源 |
|------|------|------|
| 因子议会（第一纪元） | 技术现象，可预测、可控制 | 科学认知 |
| 守望部门 | 维度边界状态指标，异常=危险 | 监控数据 |
| 拾荒者议会 | 不知道原理，只知道"有潮汐时产出高" | 经验主义 |
| 宗教正统 | 神的呼吸，虔诚者得祝福，傲慢者受惩罚 | 宗教教义 |
| 深层学者 | 损坏的裂隙装置残余运作，浓度上升=灭世倒计时 | 科学推算 |
| 探宝人行会 | "潮汐？能多挖点遗迹吗？" | 纯粹实用 |
| 学者公会 | 正在研究，初步认为是"自然现象"但存疑 | 学术研究 |
| 深层学者后继者 | 与祖先相同，但更加绝望（因为情况在恶化） | 秘密传承 |
| 普通大众 | 世界的呼吸，自然之理，无需理解 | 文化传承 |
