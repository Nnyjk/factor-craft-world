# Factor Craft 历史人物志

> **说明**：以下人物基于历史碎片推断整理。带"※"标记的人物为学界争议较大者，带"△"标记的为传说色彩浓厚、可能存在虚构成分者。

---

## 第一纪元人物

### 【因子议会核心成员】

---

#### 艾瑟恩·源质（Essence of Origin）※

| 项目 | 内容 |
|------|------|
| **生卒年** | FC -100 至 FC 4520（传说享年4620岁，实际存疑） |
| **身份** | 因子议会初代首席学者、因子技术之父 |
| **所属派系** | 萃取部门（后扩展至全议会） |
| **历史地位** | 第一纪元真正的奠基者 |

**人物概述**：

艾瑟恩·源质是第一个成功从虚空抽取能量并将其凝结为可用因子的人类。关于他的出身已无可靠记载，只有化石碎片显示他出身于"第三学术区"——当时尚属边缘的学者社群。

他一生致力于证明"知识是文明的根基"，的化石语录在第一纪元广泛流传。然而讽刺的是，正是他开创的因子技术，最终毁灭了他所建立的一切。

**化石语录**：

> "虚空是无尽的，但它不是永恒的。学会索取之前，先学会节制。"
> 
> ——艾瑟恩·因子，《首卷·第三章》（残片）

**争议**：

有学者指出，艾瑟恩晚年的部分化石语录与早期风格不符，怀疑部分"语录"为后世伪造。但更激进的学者认为，艾瑟恩本人可能早已预见到因子技术的危险性，并在晚年试图留下警告——只是这些警告被议会中的"发展派"删除或篡改。

**※ 关键争议**：艾瑟恩是否预见了灾变？

### 【归本计划核心人物】

---

#### 卡尔丁·裂隙（Karding Rift）※

| 项目 | 内容 |
|------|------|
| **生卒年** | FC 4400 至 FC 4500 |
| **身份** | 因子议会第十七任议长、归本计划主要推动者 |
| **所属派系** | 议会发展派 |
| **历史地位** | 第一纪元崩溃的直接责任人 |

**人物概述**：

卡尔丁·裂隙是因子议会历史上最具争议的议长。他出身于萃取部门，凭借对因子技术极限的执着追求一路升迁至议会最高位。

在他的领导下，议会逐渐形成一个共识：**因子的使用还未触及极限**，而"归本计划"——将因子逆向转化回虚空——将为文明打开一扇通向无限能源的大门。

他是一个相信"进步不可阻挡"的乐观主义者，也是一个对反对声音缺乏耐心的政治强人。当反对派学者提交警告报告时，他以"阻碍文明进步"为由将其压制。

**化石语录**：

> "历史告诉我们，每一次对未知的恐惧最终都会被证明是多余的。我们不是在打开一扇门，我们是在打开整个宇宙。"
>
> ——卡尔丁·裂隙，归本计划启动仪式致辞（残片）

**争议**：

有化石碎片显示，部分议会内部通讯暗示卡尔丁可能在灾变前数十年就已知道归本计划的风险。但这些碎片真伪难辨，有学者认为这是后世对"失败者"的污名化。

**※ 关键争议**：卡尔丁是"不知者无罪"还是"知其不可而为之"？

#### 奥伯恩·反向（Oberon Reverse）※

| 项目 | 内容 |
|------|------|
| **生卒年** | FC 4420 至 FC 4300（议会否决前自杀） |
| **身份** | 因子议会首席科学顾问、归本计划反对派领袖 |
| **所属派系** | 议会保守派 |
| **历史地位** | 第一纪元最孤独的警告者 |

**人物概述**：

奥伯恩·反向是萃取部门的首席科学家，也是艾瑟恩·源质的最后一位在世弟子。他从师承中继承了导师对"知识边界"的敬畏，也因此成为第一个系统论证"归本计划危险性"的学者。

他在议会听证会上提交了一份长达三百片的化石报告，逐条论证逆向转化可能导致维度边界崩溃。报告被议会以"证据不足"为由否决。

此后，奥伯恩开始了长达五十年的"沉默抗议"——他不再公开发表言论，只是不断在化石上记录自己的担忧。这些记录后来被称为"反向化石"，成为研究第一纪元崩溃的核心文献。

**化石语录**：

> "他们在建造一艘驶向深渊的船，而船上的人以为那是通往天堂的道路。"
>
> "我不是在反对进步，我是在说，有些进步是不可逆转的，而不可逆转的进步必须在开始之前就三思。"
>
> ——奥伯恩·反向，"反向化石"系列（残片）

**争议**：

奥伯恩在议会表决前自杀身亡。官方记录称其"因学术压力过大"，但有学者认为他"以死明志"——用自己的生命为反对票背书。更有人认为他是被"发展派"暗杀，但缺乏证据。

**※ 关键争议**：奥伯恩的警告化石中是否藏有关闭裂隙装置的方法？他的死是自杀还是他杀？

#### 赛琳娜·因子（Selina Factor）

| 项目 | 内容 |
|------|------|
| **生卒年** | FC 4450 至 FC 4490 |
| **身份** | 守望部门维度监控主管 |
| **所属派系** | 守望部门（中立技术部门） |
| **历史地位** | 第一个系统记录维度边界异常的人 |

**人物概述**：

赛琳娜·因子是守望部门最优秀的监控专家。她主持设计的维度监控系统在FC 4480年发现了维度边界的异常变化——边界开始出现肉眼不可见的"微裂隙"。

她向议会提交的紧急报告被列为"低优先级"。卡尔丁议长的批注写道："守望部门的职责是监控，不是制造恐慌。"

赛琳娜在FC 4490年的一次因子泄漏事故中受伤身亡，距归本计划启动还有十年。她的女儿后来成为第二纪元"深层学者"传统的奠基人之一。

**※ 遗产**：赛琳娜的监控日志副本在第二纪元被发现，成为证明"第一纪元政府掩盖真相"的关键证据。

### 【灾变见证者】

---

#### 无名避难所所长（Commander of the Unnamed Shelter）

| 项目 | 内容 |
|------|------|
| **生卒年** | FC 4470 至 2E 100 |
| **身份** | 深层避难所第一任所长 |
| **所属派系** | 无（避难所管理） |
| **历史地位** | 第一纪元最后的火种守护者 |

**人物概述**：

他的名字在任何现存的化石或口述传承中都没有记载。后世只知道他被幸存者们称为"守门人"——因为他坚持在避难所关闭之前接纳了所有能到达的人，却拒绝透露自己的名字。

他在避难所中度过了灾变后的第一个百年，亲手建立了避难所的管理制度，并坚持记录外界环境的变化。当第二代幸存者成长起来后，他开始系统性地隐瞒灾变的真正原因——因为他认为，知道真相只会让幸存者陷入"无谓的恐慌"。

他在临终前将真正的化石记录（来自原纪元的科学记载）交给了一个信得过的副手，要求其"永远不要公开，直到有人准备好了"。

**遗言**：

> "我不知道我们是否应该知道这些。但我知道，不知道的代价是，我们会再犯同样的错误。"
>
> "也许有一天，会有人足够强大，既能承受真相，又不会重蹈覆辙。"
>
> ——无名避难所所长，临终口述（第三代幸存者转述）

**争议**：

他交给副手的"真正的化石记录"是否完整保存？那个副手的后代是谁？深层学者组织是否就是这些人的后裔？

**※ 关键争议**：避难所长是"为了保护幸存者而隐瞒真相"，还是"害怕真相的力量而封印它"？

#### 灰烬·拾荒（Ash Scavenger）△

| 项目 | 内容 |
|------|------|
| **生卒年** | 2E 50 至 2E 200 |
| **身份** | 第一批走出避难所的幸存者之一、"拾荒者"职业的奠基人 |
| **所属派系** | 拾荒者 |
| **历史地位** | 第二纪元的第一声音 |

**传说概述**：

灰烬·拾荒是避难所开放后第一个走出地下的人类。传说他踏出避难所的那一刻，看到的是被紫色因子云笼罩的天空和远处燃烧的城市废墟。

他在废墟中搜寻了三十年，将一切能带走的东西带回避难所周边。他的儿子和孙子继承了他的事业，逐渐形成一个以"拾荒"为生的职业群体。

但传说也暗示，灰烬在拾荒过程中发现了一些"不应该发现的东西"——有传言说他曾经发现过一台仍在运作的因子监控设备，并在其上看到了"不应该存在的数据"。

**※ 争议**：灰烬·拾荒是纯粹的历史人物还是象征性人物？"不应该存在的数据"是什么？

#### 焚典·学者（Burned Scholar）※

| 项目 | 内容 |
|------|------|
| **生卒年** | 2E 300 至 2E 350 |
| **身份** | 第二纪元第一个尝试破译化石的学者 |
| **所属派系** | 早期学术团体 |
| **历史地位** | 将化石解读为"神谕"的第一人 |

**人物概述**：

焚典·学者是第一个认真尝试解读《因子化石》的人。但他不是科学家，而是一个宗教祭司。

他将化石上的技术术语解读为"神的语言"，将因子技术描述为"先觉者与神的契约"。他的解读体系成为后来《第一圣典》的基础。

**争议**：

焚典是"误读"还是"有意误导"？有学者指出，他对部分化石的解读过于刻意，似乎在有选择性地忽略那些不支持"神迹"叙事的内容。

更激进的学者认为，焚典可能知道化石的真正含义，但为了维护社会稳定而故意将其神化——毕竟，如果人们知道灾变是技术失误而非天罚，他们可能会拒绝使用一切遗迹技术。

**※ 关键争议**：焚典·学者是"无知的误读者"还是"有意的篡改者"？

### 【繁盛期人物】

---

#### 探源·院长（Dean of the Origin Institute）

| 项目 | 内容 |
|------|------|
| **生卒年** | 2E 2300 至 2E 2450 |
| **身份** | 探源院第二任院长 |
| **所属派系** | 探源院 |
| **历史地位** | 发现"灾变真相"的第一人 |

**人物概述**：

探源·院长是第二纪元最杰出的学者之一。在他的领导下，探源院发展出一套系统的化石解读方法，并将解读范围扩展到非宗教领域。

在他的第五十年任期中，探源院发现了一批来自第一纪元的"实验记录"化石——这些化石详细记载了归本计划从提出到被否决的全过程，以及奥伯恩·反向的警告报告。

探源·院长在发现这些化石的第三天召开了一次秘密会议。据与会者回忆，他在会上只说了一句话：

> "我们不是第一代文明。我们是第二，而且我们已经知道了第一是怎么灭亡的。"

他的这次发现直接促成了"深层学者"组织的成立。但三年后，他在一次"学术交流"中遭遇意外身亡。

**争议**：

探源·院长的死是意外还是谋杀？他是死于"意外"还是被"宗教势力"或"政治势力"清除？

#### 裂隙·守望（Rift Watcher）

| 项目 | 内容 |
|------|------|
| **生卒年** | 2E 2500 至 2E 2700 |
| **身份** | 深层学者组织第一任核心成员 |
| **所属派系** | 深层学者 |
| **历史地位** | 深层学者组织实际创始人 |

**人物概述**：

裂隙·守望是探源·院长的关门弟子。院长死后，他带着院长的核心发现和一支忠诚的学生团队转入地下，正式建立了"深层学者"组织。

他继承了院长的遗志，同时比院长走得更远——他不仅研究灾变的真相，还开始研究"如何阻止下一次灾变"。

他留下了大量的秘密化石文献，记载了第一纪元和第二纪元裂隙装置的位置和状态，以及他本人对关闭这些装置的初步构想。

**关键化石**：

> "第一纪元的裂隙装置并未全部损坏。其中至少有三座仍处于低功率运转状态。这解释了为什么因子浓度在持续上升。"
>
> "如果我们什么都不做，那么灾变不是'可能重复'，而是'必然重复'。问题是，这一次不会有人幸存。"
>
> ——裂隙·守望，秘密化石文献（残片）

**遗产**：深层学者组织在其逝世后继续存在，并延续至今（第三纪元）。

#### 教宗·天罚（Pope Divine Punishment）

| 项目 | 内容 |
|------|------|
| **生卒年** | 2E 2600 至 2E 3000 |
| **身份** | 官方宗教最高领袖 |
| **所属派系** | 宗教势力 |
| **历史地位** | 压制真相、维护"天罚叙事"的核心人物 |

**人物概述**：

教宗·天罚是《第一圣典》传统最坚定的捍卫者。当深层学者开始在秘密圈子里传播"灾变是技术失控而非天罚"的观点时，他发起了全面的压制。

在他任内，"学术管制法"被颁布，深层学者被宣布为"异端"，部分核心成员被放逐。

**争议**：

教宗·天罚是"真诚的信仰者"还是"知情的维护者"？有证据显示，他可能曾经读过深层学者的秘密文件——但他选择将其定性为"恶魔的诱惑"。

**化石语录**：

> "灾变是神对先觉者傲慢的惩罚。接受这一事实，就是接受人的有限性；否认这一事实，就是重蹈覆辙。"
>
> ——教宗·天罚，《圣典注释·卷三十七》

### 【灾变期人物】

---

#### 最后·记录者（The Last Chronicler）

| 项目 | 内容 |
|------|------|
| **生卒年** | 2E 4480 至 2E 4500 |
| **身份** | 探源院档案管理员 |
| **所属派系** | 深层学者 |
| **历史地位** | 第二纪元最后一位关键记录者 |

**人物概述**：

第二纪元灾变爆发时，最后·记录者正在探源院总部的地下档案室工作。灾变摧毁了地面建筑，但档案室的防护结构让它幸免于难。

他在档案室中度过了灾变最激烈的七天。在这七天里，他用尽一切手段将核心档案转移到更深的地下避难所。最终，他在一次余震中被埋。

他留下的最后一条记录是：

> "档案已转移。核心文献已复制三份，分别存放于三个地点。密钥已交给信任的人。"
>
> "如果有人读到这条记录，请找到深层学者的后继者。告诉他们：真相已经保存，但时机还没有到。"
>
> "等那个能够承受真相的人出现。"
>
> ——最后·记录者，绝笔

**争议**：

他转移的三份核心文献在哪里？密钥交给了谁？"能够承受真相的人"是谁？
